# 🕒 Digital Clock

A simple Python GUI application that displays the current time and date using Tkinter.

## ✨ Features

- Live Digital Clock
- Displays Current Time
- Displays Current Day, Date, Month, and Year
- Automatically Updates Every Second

## 🛠️ Technologies Used

- Python 3
- Tkinter

## 📂 Project Structure

```
Digital_Clock/
│── Project.py

```

## ▶️ How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Run the following command:

```bash
python digital_clock.py
```
