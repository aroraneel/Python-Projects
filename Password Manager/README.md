# 🔐 Password Manager

A simple Python application that allows users to save, view, and generate passwords.

## ✨ Features

- Save Website Passwords
- View Saved Passwords
- Generate Random Passwords
- Store Passwords in a Text File
- Exit Program

## 🛠️ Technologies Used

- Python 3

## 📂 Project Structure

```
Password_Manager/
│── Project.py
│── about.txt
│── README.md
```

## ▶️ How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Run the following command:

```bash
python password_manager.py
```

## 📚 Concepts Used

- Dictionary
- File Handling
- Functions
- Random Module
- String Module
- While Loop
- If-Else Statements

## ⚠️ Note

This project is created for learning purposes only.
Passwords are stored in plain text and are **not encrypted**.
