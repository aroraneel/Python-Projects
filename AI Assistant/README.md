# 🤖 AI Assistant

A beginner-friendly AI Assistant built using Python, LangChain, LangGraph, and Anthropic's Claude model. The assistant can understand natural language, answer questions, and use custom tools to perform specific tasks.

## ✨ Features

- Interactive AI Chat
- Calculator Tool
- Greeting Tool
- AI Tool Calling
- Environment Variable Support
- Streaming AI Responses
- Easy to Extend with New Tools

## 🛠️ Technologies Used

- Python 3
- LangChain
- LangGraph
- Anthropic Claude API
- Python Dotenv

## 📚 Concepts Used

- AI Agents
- Large Language Models (LLMs)
- Tool Calling
- Prompt Processing
- Functions
- Environment Variables
- While Loops
- Python Decorators
- Streaming Responses

## ▶️ How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/your-repository.git
```

### 2. Navigate to the project folder

```bash
cd your-repository
```

### 3. Install the required packages

```bash
pip install -r requirements.txt
```

### 4. Create a `.env` file

```text
ANTHROPIC_API_KEY=your_api_key_here
```

### 5. Run the application

```bash
python main.py
```
