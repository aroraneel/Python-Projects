# 🎓 Student Result Manager

A simple Python project to manage student records using a menu-driven interface.

## ✨ Features

- Add Student
- View All Students
- Check Student Result (Pass/Fail)
- Exit Program

## 🛠️ Technologies Used

- Python 3

## 📂 Project Structure

```
Student_Result_Manager/
│── Project.py
│── about.txt
│── README.md
```

## ▶️ How to Run

1. Open the project in VS Code.
2. Open the terminal.
3. Run the following command:

```bash
python Student_Result_manager.py
```

- Dictionary
- While Loop
- For Loop
- If-Else Statements
- User Input
